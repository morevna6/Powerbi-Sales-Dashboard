# Sales Dashboard (Power BI)

## Project Overview

This project presents a Power BI dashboard analyzing supermarket sales data.

The dashboard focuses on understanding revenue trends, product performance, and payment method distribution. It also includes a dynamic currency conversion feature.

---

## Objectives

* Analyze total revenue and sales trends
* Compare product categories
* Understand customer payment behavior
* Provide an interactive reporting experience

---

## Key Metrics

* Total Revenue
* Average Product Price
* Sales by Category
* Sales by Payment Method

---

## Dashboard Features

* Time series analysis of sales performance
* Category-based product distribution
* Payment method breakdown
* Dynamic currency selection (TRY / USD)
* Interactive filters (Year, Month)

---

## Tools Used

* Power BI

## Key Observations

* Revenue fluctuates across months, indicating possible seasonality
* Product categories show relatively balanced distribution
